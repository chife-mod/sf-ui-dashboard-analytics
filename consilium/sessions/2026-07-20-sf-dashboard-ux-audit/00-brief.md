## Задача

Обстоятельный UX/UI-аудит дашборда глазами конечного клиента (маркетолог/владелец
клиники, не аналитик): понятность, полезность, что улучшить, что переработать
полностью, проблемы по приоритетам.

## Техноты сессии

- Файл в ChatGPT заведён программно: postMessage-мост (localhost CORS-сервер →
  попап → ArrayBuffer) + `input.files = DataTransfer.files` + `change`-событие.
- Файл в Gemini: синтетические drop/paste отвергаются (isTrusted-фильтр);
  сработала связка «реальный клик по композеру + clipboard.write PNG + реальный
  Cmd+V» — при этом Gemini дофлашил ранее сброшенный в dropzone PDF (чип
  «PDF Dashboard Analysis» появился после первого настоящего жеста вставки).
- Оба треда — обычные (не временные): чип-флаш в temporary chat не проверялся.
  Гигиена: треды «Pong» (канарейка) и оба аудит-треда можно удалить из историй.
