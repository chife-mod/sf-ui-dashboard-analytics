# GPT — интеграция Opportunity-слоя в Answers-first

**Ключевое концептуальное решение:** «Окна возможностей» — не пятый вопрос
исследования, а вывод более высокого уровня, который система строит на
пересечении всех четырёх ответов (В1 — размер/динамика; В2 — барьеры/отношение;
В3 — незакрытые сценарии; В4 — конкурентное/гео-поле). Не превращать каждый
ответ в поиск окна — два уровня:

1. **В каждом ответе** — строка «сигнал возможности» (перед рекомендацией):
   - В1: сигнал не подтверждён (активность стабильна, спрос не отделён от клиник);
   - В2: ранний сигнал — доверие через прозрачность;
   - В3: потенциальное окно — долгосрочное сопровождение;
   - В4: окно пока не определено (нет geo-данных).
   Последовательность карточки: ответ → доказательства → сигнал → рекомендация.
2. **Синтетический блок «Окна возможностей»** — сверху компактная строка под
   summary-карточками («Обнаружено: 1 перспективное окно · 1 ранний сигнал ·
   2 области требуют данных») с главной карточкой окна, и полноценная секция
   перед методологией (максимум три окна, ранжированные по силе доказательств).

**Карточка окна — семь элементов:** что за окно · незакрытая потребность ·
какие данные обнаружили · почему сейчас · почему может закрыться · что может
сделать клиент · насколько уверены. Ключевой вопрос: «почему через два года
эта возможность может быть менее ценной?» (для сопровождения: «если крупные
клиники сделают годовые программы стандартом — отличие исчезнет»).

**Статусы без псевдоточности** (не «Score 87/100»): Подтверждённое окно /
Перспективное окно / Ранний сигнал / Недостаточно данных. Отдельно срочность:
Действовать сейчас / Проверить в ближайшие месяцы / Наблюдать / Сначала собрать
данные.

**Три окна для текущей темы:** долгосрочный уход после пересадки (перспективное
окно — уход фрагментирован → программа 9–12 мес) · контент о рисках и
восстановлении (ранний сигнал → transparent recovery + консультационная воронка)
· позиционирование УА-клиник против зарубежных (недостаточно данных → сначала
geo-intent).

**Позиционирование:** «Дашборд не только отвечает на исследовательские вопросы,
но и показывает, какие изменения рынка можно превратить в преимущество — пока
это не сделали конкуренты».
